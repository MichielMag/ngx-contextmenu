# Installation and setup

`@perfectmemory/ngx-contextmenu` is built on top of the `@angular/cdk` so you have to install both

## Installation

```shell
npm install @perfectmemory/ngx-contextmenu @angular/cdk
```

```shell
yarn add @perfectmemory/ngx-contextmenu @angular/cdk
```

## Compatibility matrix

| Angular   | @perfectmemory/ngx-contextmenu |
| --------- | ------------------------------ |
| `^12.0.0` | `^6.0.0`                       |
| `^13.0.0` | `^7.0.0`                       |

## Project setup

In your Angular project, you have to import:

1. Angular CDK overlay's CSS file
2. `ContextMenuModule` in your Angular application module.

### Import the Angular CDK overlay CSS

You can import it either in the main `styles.scss`, for example `./src/styles.scss`

```css
@import '@angular/cdk/overlay-prebuilt.css';
```

Or in the `angular.json`

```json
{
  ...
  "architect": {
    "build": {
      "options": {
        "styles": [
          "src/styles.scss",
          "@angular/cdk/overlay-prebuilt.css"
        ],
      },
    },
  },
  ...
}

```

### Import `ContextMenuModule` in your Angular application module

```ts
import { ContextMenuModule } from '@perfectmemory/ngx-contextmenu';

@NgModule({
  ...
  imports: [
    ContextMenuModule,
  ],
  ...
})
export class AppModule {}
```
